## Usage
The quickest way to get started is to run the following command in your Python environment:

```bash
pip install llm-vm
```

## Alternative Installation Methods

### Cloning the Repository

You can also install the LLM-VM by cloning this repository and running the setup script. Here are the steps:

```bash
git clone https://github.com/anarchy-ai/LLM-VM.git
cd LLM-VM
./setup.sh
```

Please note that the above bash script `setup.sh` works for MacOS and Linux.

### Manual Installation (Linux and MacOS)

For manual installation on Linux and MacOS, follow these steps:

```bash
git clone https://github.com/anarchy-ai/LLM-VM.git
cd LLM-VM
python -m venv <name>
source <name>/bin/activate
python -m pip install -e ."[dev]"
```
<Note>

Make sure to replace `<name>` with your preferred name when creating a virtual environment on Linux and MacOS.
</Note>


### Windows Installation

If you are using Windows, you can choose between the following methods:

#### Method 1: PowerShell Script

Before proceeding, open PowerShell as an administrator and run the following command:

```powershell
Set-ExecutionPolicy RemoteSigned
```

Afterward, you can use the following PowerShell script to install:

```powershell
git clone https://github.com/anarchy-ai/LLM-VM.git
cd LLM-VM
.\windows_setup.ps1
```

#### Method 2: Using Python and virtualenv

Follow these steps for installation on Windows:

```powershell
winget install Python.Python.3.11
python --version
git clone https://github.com/anarchy-ai/LLM-VM.git
cd LLM-VM
python -m venv anarchyai
anarchyai\Scripts\activate
python -m pip install -e .
```
<Note>
**Note:**

- For the Windows steps to work, you need to be on Windows 10 build 16299 or later.
- Enabling developer mode in Windows settings is recommended but not compulsory, as it can provide added advantages.

</Note>
